Hello World

A program to greet the planet.

This page is an attempt to greet everyone using our various human languages. We're interested to see how long our list can become.

We'll start with the languages that the Epicodus staff could offer without using Google.

  • English: Hello, World!
  • Spanish: Hola, Mundo!
  • Japanese: Konichiwa, Sekai!
  • French: Bonjour, Monde!
  • Kinyarwanda: Mwirwe, Isi!
